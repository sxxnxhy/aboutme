나만의 카카오톡 만들기 — WebSocket 기반 채팅 서비스

  • 2025년 02월 - 2025년 03월
  • 실시간 채팅 애플리케이션 개발 및 AWS EC2 배포.
  • 기술:
    Java, Spring Boot, WebSocket (STOMP), Redis, MySQL, JPA, HTML, JavaScript, CSS, AWS EC2
  • 설명:
    WebSocket을 공부하며 카카오톡 같은 실시간 채팅 서비스를 목표로 시작한 개인 프로젝트입니다. 단순한 채팅을 넘어 채팅 목록, 읽음/안 읽음 처리, 보안, 효율적인 데이터 처리를 구현했어요. RESTful API를 설계해 프론트엔드에서 호출 가능하게 했고, Redis로 성능을 최적화하며 AWS EC2에 배포했습니다.
실제 운영 중인 서비스 주소입니다. 실제 서비스를 경험해보고 싶어서 AWS에 업로드했습니다. WebSocket은 HTTPS 환경에서만 동작하기 때문에 SSL 인증도 받았습니다.

  • Redis 기반 채팅 리스트 읽음/읽지 않음 알림 시스템
    Redis로 채팅 리스트의 읽음/읽지 않음 상태를 실시간 관리.

마치며

저는 개발자에게 코딩 실력만큼이나 중요한 건 단순히 "무엇을 했는지" 나열하는 게 아니라 "왜 했는지", "어떻게 했는지", "무엇을 배웠는지"를 보여주는 과정, 문제를 정의하고 해결하는 힘이 중요하다고 생각합니다.

결국 개발도 사람들이 하는 거기에 결과만큼 중요한 건 이 과정에서 함께하는 팀과 명확히 전달하는 커뮤니케이션 능력이 정말 중요하다고 생각합니다.

팀 프로젝트에서 팀원을 이끌고, 문제를 해결하며 배운 점을 블로그에 기록한 경험은 이런 가치를 깨닫게 했습니다.

앞으로도 기술적 성장뿐 아니라 소통을 통해 함께 발전하는 개발자가 되기 위해 끊임없이 노력하겠습니다. 제 포트폴리오를 읽어봐주셔서 정말 감사합니다!
